# 🤖 AI Doubt Solver — Intelligent Learning Assistance Platform

AI Doubt Solver is an AI-powered academic assistant designed to help students resolve doubts instantly using advanced artificial intelligence. The platform supports text, image, and voice-based questions across multiple academic domains including school education, higher education, and competitive exams.

This project demonstrates the integration of modern AI technologies with a scalable full-stack architecture to create a personalized learning experience.

# ⚙️ Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/ai-doubt-solver.git
cd ai-doubt-solver
```

## 2️⃣ Install Dependencies

Frontend:

```bash
cd frontend
npm install
```

Backend:

```bash
cd backend
npm install
```

## 3️⃣ Environment Variables

Create `.env` file in backend:

```
OPENAI_API_KEY=your_api_key_here
DATABASE_URL=your_database_url
JWT_SECRET=your_secret_key
```

## 4️⃣ Run Application

Backend:

```bash
npm run server
```

Frontend:

```bash
npm run dev
```

Application will run on:

```
http://localhost:3000
```
